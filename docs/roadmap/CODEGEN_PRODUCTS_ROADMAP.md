# Codegraph 엔진 기반 제품 로드맵

> IR + CFG + DFG + 타입추론 인프라 활용 제품/기능 (임팩트순 정렬)

---

## 🏆 Tier 1: 높은 임팩트 + 빠른 구현 (1-2주)

### 1. 코드 난독화 도구 ✅ 구현중
- **상태**: 기본 완료
- **기능**: 식별자 난독화, Docstring 제거, 타입힌트 변환, Nuitka 연동
- **남은 것**: CLI, 자동 exclude 추천, 문자열 암호화
- **경쟁**: PyArmor ($56-512)
- **예상 가격**: $40-100/년

### 2. 보안 스캐너 (SAST)
- **상태**: 엔진 완성, 제품화 필요
- **기능**: SQL Injection, XSS, Command Injection, 시크릿 탐지, CVE 매칭
- **출력**: SARIF, GitHub/GitLab CI 연동
- **경쟁**: Snyk ($25/user), SonarQube ($150+)
- **예상 가격**: $50-200/월

### 3. 코드 검색 CLI
- **상태**: MCP로 일부 구현
- **기능**: 자연어 검색, 시맨틱 검색, 유사 코드 검색
- **경쟁**: Sourcegraph ($49/user)
- **예상 가격**: $10-30/월

### 4. 데드코드/미사용 import 제거
- **상태**: IR로 바로 가능
- **기능**: 안 쓰는 함수/클래스/import 자동 삭제
- **예상 개발**: 3-5일

### 5. LLM 컨텍스트 최적화
- **상태**: 일부 구현
- **기능**: 질문에 필요한 코드만 추출, 토큰 절약 (30-50%)
- **가치**: API 비용 절감

---

## 🥈 Tier 2: 높은 임팩트 + 중간 구현 (2-4주)

### 6. 코드 리뷰 봇
- **기능**: PR 자동 분석, 보안/스타일/복잡도 코멘트, 리뷰어 추천
- **연동**: GitHub, GitLab, Bitbucket
- **경쟁**: CodeRabbit ($15/월)
- **예상 가격**: $10-20/월

### 7. 자동 리팩토링 도구
- **기능**: Rename, Extract Method/Variable/Class, Inline, 타입 추가
- **경쟁**: Sourcery ($10/월)
- **예상 가격**: $10-15/월

### 8. 마이그레이션 도구
- **기능**: Flask→FastAPI, Django→FastAPI, Python 2→3, Callback→Async
- **경쟁**: 없음 (블루오션)
- **예상 가격**: $50-200/프로젝트

### 9. 기술 부채 대시보드
- **기능**: 복잡도, 중복, 커버리지 → 종합 점수, 트렌드, 비용 환산
- **경쟁**: CodeClimate, CodeScene
- **예상 가격**: $20-50/월

### 10. 의존성/아키텍처 분석
- **기능**: 의존성 그래프, 순환 탐지, 레이어 위반, 영향도 분석
- **시각화**: 인터랙티브 그래프, 히트맵
- **예상 개발**: 2-3주

### 11. API 문서 자동 생성
- **기능**: IR → OpenAPI 스펙, Docstring 추출, 예제 생성
- **예상 개발**: 2주

## 🥉 Tier 3: 중간 임팩트 (1-2개월)

### 12. 테스트 자동 생성
- **기능**: 함수 시그니처 → 테스트 스켈레톤, 경계값, Mock 생성
- **경쟁**: Diffblue (Java, 고가)
- **예상 개발**: 4-6주

### 13. IDE 플러그인
- **대상**: VS Code, JetBrains (PyCharm/IntelliJ)
- **기능**: 인라인 분석 결과, 리팩토링, 검색
- **예상 개발**: 4-6주

### 14. 디버깅/포렌식 도구
- **기능**: 버그 원인 추적, 예외 전파 분석, 데드락/레이스 컨디션 탐지
- **예상 개발**: 3-4주

### 15. 성능 분석
- **기능**: N+1 쿼리 탐지, O(n²) 패턴, 캐시 기회, 병렬화 기회
- **예상 개발**: 3-4주

### 16. 코드 품질 자동화
- **기능**: 매직넘버 추출, 네이밍 개선, 함수 분리 추천, 타입 커버리지
- **예상 개발**: 2-3주

### 17. 컴플라이언스/규제 검사
- **기능**: GDPR, SOC2, PCI-DSS, HIPAA 패턴 체크
- **예상 개발**: 3-4주

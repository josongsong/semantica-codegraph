# Release Notes

본 문서는 Semantica 버전별 주요 변화(high-level change log)를 기록한다.
세부 기술 스펙이나 구현 변경은 기록하지 않는다.

---

## 목적

- 버전별 주요 변화 요약
- 팀/조직 내 공유용 기록
- 기능 추가/변경/폐기 내역의 상위 수준 설명

---

## 작성 규칙

- MUST: 기술 상세/코드 상세는 포함하지 않는다
- MUST: 하위 시스템 변경은 high-level로만 기록
- SHOULD: 각 항목에 날짜/요약/영향 범위 포함
- MUST NOT: specification/standards 내용을 중복 기술하지 않는다

---

## v2 (현재 버전)

- 문서 레이어 전체 구조 정립
- overview/specification/standards/knowledge/ops/domain/decisions 구분 확립
- Codegraph 엔진 스펙 구조 수립
- 초기 인덱싱·검색·그래프 스펙 정리
- API/DTO/Chunk/Schema 기준 정립

---

## v3 (예정)
